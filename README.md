### How to setup and compile

##### 1. Clone the repository
- Install Flutter version `3.22.2`

##### 2. Clone the repository
```bash
git clone https://github.com/LaPyaeOo/chat
```

##### 2. To run the app, simply write
```bash
flutter clean
flutter pub get
flutter run
```
